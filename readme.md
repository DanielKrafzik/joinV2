# Join – Task Management App

Join is a modern Kanban-style task management application that helps users organize, track, and manage their tasks efficiently. It focuses on a clean user interface, intuitive interactions, and a structured workflow.

---

## 🌐 Live Demo

👉 <a href="https://join-app-dk.web.app" target="_blank">
  Live Demo
</a>

---

## ✨ Features

* Create, edit, and delete tasks
* Drag & drop tasks between columns
* Organize tasks with categories and priorities
* Responsive design for desktop and mobile
* Clean and user-friendly interface
* Data persistence with Firebase

---

## 🛠️ Tech Stack

* Angular
* TypeScript
* HTML5
* CSS3
* Firebase

---

## ⚙️ Installation

To run the project locally:

```bash
git clone https://github.com/DanielKrafzik/join.git
cd join
npm install
ng serve
```

Then open:
http://localhost:4200
